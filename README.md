# Railway GCC Contract Analyzer

A FastAPI + React JS application that analyzes Railway contract PDFs against the Indian Railways General Conditions of Contract (GCC) standard clause library using Gemini AI.

---

## Backend Setup

```bash
cd backend

# Create and activate virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install Tesseract OCR (macOS)
brew install tesseract

# Set up environment variables
cp .env.example .env
# Edit .env and add your GEMINI_API_KEY

# Generate the GCC clause reference JSON
# With an official GCC PDF:
python scripts/extract_gcc_clauses.py --pdf path/to/gcc.pdf

# Or use the built-in seed data (no PDF required):
python scripts/extract_gcc_clauses.py

# Start the FastAPI server
uvicorn main:app --reload --port 8000
```

The API will be available at `http://localhost:8000`.
API docs: `http://localhost:8000/docs`

## API Endpoints

### `GET /api/health`
Returns server status and number of GCC clauses loaded.

### `POST /api/analyze`
Accepts a PDF upload and returns structured clause mapping results.

**Request:** `multipart/form-data` with field `pdf` (the contract PDF file).

**Response:**
```json
{
  "filename": "contract.pdf",
  "page_count": 50,
  "total_gcc_clauses": 12,
  "found_count": 9,
  "missing_count": 3,
  "part_errors": [],
  "clauses": [
    {
      "clause_id": "GCC-47",
      "clause_title": "Termination by Employer",
      "risk_category": "HIGH",
      "status": "present",
      "page_number": 23,
      "confidence": "high",
      "verbatim_text": "...",
      "matched_keywords": []
    }
  ]
}
```

**Clause status values:**
- `present` — found and confirmed by Gemini with high/medium confidence
- `present_fuzzy` — not found by Gemini but confirmed by fuzzy keyword search
- `truly_missing` — not found by either method; likely absent from the contract

## Project Structure

```
backend/
├── main.py                        # FastAPI entrypoint
├── requirements.txt
├── .env.example
├── data/
│   └── gcc_clauses.json           # Generated by extract_gcc_clauses.py
├── scripts/
│   └── extract_gcc_clauses.py     # One-time GCC extraction script
└── core/
    ├── __init__.py
    ├── pdf_extractor.py           # PyMuPDF + OCR text extraction
    ├── gemini_mapper.py           # Gemini 3-part mapping pipeline
    └── clause_matcher.py          # Anchor extraction + fuzzy fallback
```
